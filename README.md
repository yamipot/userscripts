# <img src="./icon.svg" width="42" alt="ehpeek panda icon" align="absmiddle"> ehpeek

Another reader for E-H / ExH galleries. Optimized for touchscreen and mobile.

## Features

- Fullscreen reader with vertical-scroll and page-flip modes.
- Touch-friendly UI enhancement: swipe navigation, larger button...
- Best-effort compatibility with:
  - [EhSyringe](https://github.com/EhTagTranslation/EhSyringe).

## Install

- Install [Tampermonkey](https://www.tampermonkey.net/). On Android, it can be used with [Quetta](https://www.quetta.net/products/android), which supports browser extensions.
- Install the userscript from one of these sources:
  - [GitHub](https://github.com/yamipot/ehpeek/raw/build-master/ehpeek.user.js)
